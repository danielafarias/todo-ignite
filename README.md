> Todo-Ignite é uma aplicação de tarefas a fazer. Desafio para aulas de Fundamentos de ReactJS da Rocketseat Trilha Ignite.

### ❤️ Ajustes e melhorias

O projeto ainda está em desenvolvimento e as próximas atualizações serão voltadas nas seguintes tarefas:

- [x] Projeto inicializado
- [x] Criar uma tarefa
- [x] Alterar status de uma tarefa
- [x] Remover uma tarefa
- [x] Projeto Finalizado

## 💻 Tecnologias

Foram usadas as seguintes tecnologias:

- ReactJS;
- Webpack;
- Babel;
- Sass;

## 🚀 Instalando `<Todo-Ignite>`

Para instalar o `<Todo-Ignite>`, siga estas etapas:

Clone o projeto:
```
<git clone https://github.com/danielafarias/todo-ignite.git>
```
Em seguida, adicione as bibliotecas e dependências necessárias:
```
<cd Todo-Ignite>
```
```
<yarn>
```

## 🧑‍💻 Usando `<Todo-Ignite>`

Para usar `<Todo-Ignite>`:

```
<yarn dev>
```
